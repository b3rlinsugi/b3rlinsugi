## 👨‍💻 About Me

Hello! I'm **Berlin Sugiyanto**, a passionate Backend Developer and recent Informatics Engineering graduate from Universitas Gunadarma. I hold an official **BNSP Certification as a Junior Web Developer**, demonstrating my commitment to national industry standards.

My core expertise lies in designing scalable **RESTful APIs**, implementing secure **Authentication Systems** (JWT, RBAC), and architecting robust **Database Schemas**. While my primary focus is backend engineering, I also build modern, full-stack applications using the latest frontend technologies to deliver seamless user experiences.

- 🔭 &nbsp;**Focus Area**: System Architecture, Secure API Development, Database Optimization.
- 🌱 &nbsp;**Currently Mastering**: Next.js 15, Turbopack, Edge Functions, and Advanced Server-Side State Management.
- 🎓 &nbsp;**Education**: S1 Informatics Engineering — Universitas Gunadarma (GPA: 3.63).
